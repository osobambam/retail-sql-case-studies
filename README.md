# SQL Business Analytics Portfolio

A collection of SQL case studies built in **Microsoft SQL Server**, each designed around a real business question — not a tutorial exercise.

Every project follows the same structure: a business problem is defined, data is queried to surface evidence, and a recommendation is made based on what the data actually shows — including when the findings challenge the original hypothesis.

---

## Structure

Each project folder contains:
- A `README.md` with the full case study — business context, schema, findings and recommendation
- A `schema/` folder with table creation scripts
- A `queries/` folder with individual `.sql` files, one per business question
- A `output/` folder with screenshots of actual query results
